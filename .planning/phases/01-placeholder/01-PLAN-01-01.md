# PLAN-01-01: Заглушка и сборка

**Wave:** 1
**Depends on:** none
**Autonomous:** yes
**Requirements:** SETUP-01, STUB-01, BUILD-01

## Objective

Сделать проект с Vite + Tailwind, сверстать страницу-заглушку, проверить dev-сервер и production-сборку.

## must_haves

- `npm run dev` открывает страницу с заглушкой в браузере
- `npm run build` собирает `dist/` без ошибок
- Страница содержит имя, «Сайт скоро откроется», email-контакт
- Русский текст без кракозябр
- Страница центрирована, адаптивна, стилизована (Tailwind)

## Task 2: Верстка страницы-заглушки

<read_first>
- file: .planning/phases/01-placeholder/01-UI-SPEC.md (layout, typography, colors, spacing)
- file: .planning/phases/01-placeholder/01-CONTEXT.md (decisions D-05, D-06 — контент и структура)
- file: .planning/phases/01-placeholder/SKELETON.md (CSS custom properties)
</read_first>

<acceptance_criteria>
- `index.html` содержит: `<meta charset="UTF-8">`, `<meta name="viewport" ...>`, `<link>` на Google Fonts Inter
- `index.html` содержит три секции: `<header>` (имя), `<main>` (сообщение + контакт), `<footer>` (копирайт)
- `src/style.css` содержит: `@import "tailwindcss"`, CSS custom properties в `:root`
- Страница использует Tailwind utility classes для центрирования (`flex min-h-screen items-center justify-center`)
- При `npm run dev` страница отображается: заголовок, «Сайт скоро откроется», email-ссылка, футер
</acceptance_criteria>

<action>
1. Создать `index.html` (в корне проекта — требование Vite vanilla):
   - `<meta charset="UTF-8">` в `<head>`
   - `<link rel="preconnect" href="https://fonts.googleapis.com">`
   - `<link href="https://fonts.googleapis.com/css2?family=Inter:wght@400;600;700&display=swap" rel="stylesheet">`
   - `<script type="module" src="/src/main.js"></script>` (пустой main.js)
   - Секции:
     - `<header>`: `<h1 class="text-4xl ...">` с именем (плейсхолдер `[Имя Фамилия]`)
     - `<main>`: `<p class="text-xl ...">Сайт скоро откроется</p>`, `<a href="mailto:...">` с email-ссылкой
     - `<footer>`: `<p class="text-sm ...">© 2026 [Имя Фамилия]</p>`
   - Tailwind классы по UI-SPEC: `flex flex-col min-h-screen items-center justify-center gap-8 p-6`
2. Создать `src/style.css`:
   - `@import "tailwindcss";`
   - `@layer base { body { font-family: 'Inter', ...; } }`
   - `:root { --color-bg, --color-text, --color-text-secondary, --color-accent, --color-accent-hover, --font-sans }` (из SKELETON.md)
3. Создать `src/main.js` (пустой файл или `import './style.css'`)
4. Создать `public/favicon.ico` (простой файл, можно сгенерировать через emoji-favicon или взять минимальный .ico)

**Плейсхолдеры для замены:** `[Имя Фамилия]` → реальное имя, `[email@example.com]` → реальный email
</action>

## Task 3: Проверка production-сборки

<read_first>
- file: .planning/phases/01-placeholder/01-RESEARCH.md (build output expectations)
</read_first>

<acceptance_criteria>
- `npm run build` завершается кодом 0
- `dist/index.html` существует и содержит `<meta charset="UTF-8">`
- `dist/assets/` содержит минифицированный CSS-файл
- Размер CSS < 50 KB (Tailwind tree-shaking работает)
- `npm run preview` открывает собранную страницу
</acceptance_criteria>

<action>
1. Выполнить `npm run build`
2. Проверить вывод: нет ошибок, размер файлов в пределах ожидаемого
3. Проверить `dist/index.html` — содержит ссылки на собранные ассеты с хешами
4. Выполнить `npm run preview` — убедиться, что страница работает из production-сборки
</action>
