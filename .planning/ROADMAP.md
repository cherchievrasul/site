# Roadmap: Персональный сайт

**Created:** 2026-06-24
**Granularity:** coarse
**Mode:** mvp

## Phases

### Phase 1: Заглушка + Деплой
**Goal:** Сайт доступен по домену с оформленной заглушкой и проверенным пайплайном деплоя
**Mode:** mvp
**UI hint:** yes
**Requirements:** SETUP-01, BUILD-01, DEPLOY-01, STUB-01

**Success Criteria:**
1. При переходе по домену открывается оформленная страница-заглушка (не ошибка, не дефолтная страница хостинга)
2. На странице отображается имя/название, «скоро открытие» и способ связи
3. Страница корректно отображается на мобильных (320px+) и десктопе
4. Русский текст без кракозябр, страница открывается по HTTPS
5. Пайплайн деплоя (`npm run build` → FTP в `public_html/`) проверен на практике

### Phase 2: Дизайн и контент
**Goal:** Полноценный сайт-визитка — hero, about, контакты, фото, адаптивная вёрстка
**Mode:** mvp
**UI hint:** yes
**Requirements:** HERO-01, NAV-01, ABOUT-01, PORT-01, CONTACT-01, SEO-01, FOOT-01, RESP-01, ERR-01

**Success Criteria:**
1. Посетитель видит hero-секцию с именем, заголовком, фото
2. Навигация по якорным ссылкам с плавным скроллом работает на всех устройствах
3. Посетитель находит портфолио (сетку карточек) и контакты (email + соцсети)
4. Сайт полностью работоспособен на 320px, 768px, 1920px+
5. Кастомная страница 404 в стиле сайта

### Phase 3: Полировка и интерактив
**Goal:** Тёмная тема, форма связи, анимации, навыки, аналитика
**Mode:** mvp
**UI hint:** yes
**Requirements:** THEME-01, FORM-01, SKILL-01, RESUME-01, ANALYTICS-01, ANIM-01

**Success Criteria:**
1. Переключатель тёмной/светлой темы сохраняет выбор между сессиями
2. Форма связи отправляет сообщение с подтверждением доставки
3. Секция навыков + резюме с версией для печати
4. Анимации появления при скролле без рывков
5. Аналитика посещений без cookie-баннеров (privacy-first)
