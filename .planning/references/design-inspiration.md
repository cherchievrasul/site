# Референсы дизайна — персональные сайты

Собрал лучшие примеры по категориям. У каждого — что именно стоит скопировать.

## Минимализм (best fit для текущего сайта)

| Сайт | Что брать |
|------|-----------|
| [brittanychiang.com](https://brittanychiang.com) | Чистая структура: hero → about → projects → contacts. Синий акцент на белом. Плавный скролл. |
| [tobiasahlin.com](https://tobiasahlin.com) | Типографика и отступы. Шрифт без засечек, крупные заголовки, много воздуха. |
| [paco.me](https://paco.me) | Анимации при скролле. Тонкие, не отвлекающие. Intersection Observer. |
| [jacekjeznach.com](https://jacekjeznach.com) | Сетка проектов. Карточки с превью, hover-эффект. |

## Что применить прямо сейчас

Из всех референсов самое релевантное для текущего сайта:

1. **Структура brittanychiang.com** — hero → about → projects → contacts, синий акцент
2. **Сетка joshwcomeau.com** — карточки с hover-эффектом и тегами
3. **Типографика rsms.me** — крупный шрифт, воздух, 65ch ширина
4. **Анимации paco.me** — едва заметные, не отвлекающие

Скажи какое направление нравится — обновлю дизайн.
